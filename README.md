# Plus ou moins en nodejs

## Usage

```shell
# Cloner le projet sur sa machine
git clone https://github.com/rolljee/plus-ou-moins

# Installer les dépendances
cd plus-ou-moins && npm install
```

```javascript
// Lancer le programme
node index.js
```

## Comment ça marche

Nous utilisons la puissance des évènements ( grâce à la librairie Events de node )

Afin de permettre au programme de demander continuellement de rentrer un input tant qu'il n'a pas trouver le bon chiffre.
